# AI Smart Crawler - 开源贡献指南

## 🚀 如何贡献

### 1. Fork 项目
```bash
# 点击GitHub页面的 Fork 按钮
```

### 2. 克隆你的Fork
```bash
git clone https://github.com/YOUR_USERNAME/ai-smart-crawler.git
cd ai-smart-crawler
```

### 3. 创建分支
```bash
git checkout -b feature/your-feature-name
```

### 4. 提交更改
```bash
git add .
git commit -m "Add: your feature description"
git push origin feature/your-feature-name
```

### 5. 创建Pull Request
在GitHub上创建PR，描述你的更改。

---

## 📋 开发规范

### 代码风格
- 遵循 PEP 8
- 使用类型注解
- 添加 docstring

### 提交信息规范
```
Add: 新功能
Fix: 修复bug
Update: 更新功能
Docs: 文档更新
Test: 测试相关
Refactor: 重构
```
